# HCC WSI Prognosis — Whole-Slide Image Survival Analysis for HCC

## Overview | 项目简介

This repository contains the complete analysis pipeline for the paper:

> **Whole-Slide Image-Based Deep Learning Predicts Prognosis in Hepatocellular Carcinoma: A 106-Patient TCGA-LIHC Cohort Study**

We use an Attention-Based Multiple Instance Learning (ABMIL) model trained on CTransPath features from H&E-stained whole-slide images (WSI) to predict overall survival in HCC. The pipeline further integrates:
- RNA-seq differential expression and GSEA pathway analysis
- MRI arterial-phase radiomics with Cox-LASSO feature selection

本仓库包含论文的完整分析流程，涵盖：WSI深度学习生存预测（ABMIL）、转录组差异表达与通路富集（GSEA）、MRI放射组学生存模型（Cox-LASSO）。

## Quick Start | 快速开始

### 1. Install dependencies

```bash
pip install -r requirements.txt
# or with uv (faster):
uv pip install -r requirements.txt
```

### 2. Configure paths

`configs/paths.py` now prefers environment variables and falls back to project-relative defaults. 这样避免在仓库中直接写绝对路径。

示例（推荐）:

```bash
export HCC_OUT_DIR=/your/path/reports
export HCC_GDC_DIR=/your/path/gdc_downloads
export HCC_EXT_CLINICAL_CSV=/your/path/tcga_data_formatted_final.csv
export HCC_SVS_BASE_DIR=/your/path/svs_files
export HCC_MODEL_DIR=/your/path/abmil_checkpoints
export HCC_EMBEDDINGS_DIR=/your/path/ctranspath_embeddings
```

你也可以在 `configs/paths.py` 中直接修改默认值（只需编辑项目相对路径而不是硬编码 `/home/`）：

```python
OUT_DIR = _p("HCC_OUT_DIR", REPO_ROOT / "reports")
GDC_DIR = _p("HCC_GDC_DIR", REPO_ROOT / "data" / "gdc" / "gdc_downloads")
# ...其他变量...
```

**验证路径加载**:

```bash
python -c "from configs.paths import GDC_DIR, OUT_DIR; print(GDC_DIR, OUT_DIR)"
```

### 3. Run pipeline

```bash
# Step 1: Aggregate TPM matrix from GDC downloads
python preprocess/step1_aggregate_tcga_tpm.py

# Step 2: Build 106-patient risk cohort
python analysis/step2_build_risk_cohort.py

# Step 3: DEG + GSEA
python analysis/step3_deg_gsea.py

# Step 4: MRI radiomics + Cox-LASSO
python analysis/step4_mri_radiomics.py

# Figures
python figures/fig_km_heatmap.py
python figures/fig_volcano_gsea.py
python figures/fig_lasso_path.py
python figures/fig_attention_heatmap.py
```

## Methods Summary | 方法简介

### WSI Risk Stratification

- **Feature extractor**: CTransPath (ViT-based self-supervised model pre-trained on pathology images)
- **Tile size**: 512 × 512 px at 20× magnification
- **Model**: ABMIL with Cox proportional-hazards risk head
- **Training**: 5-fold cross-validation; ensemble averaged for final predictions
- **Cohort**: 101 patients with WSI; 5 additional patients supplemented via GDC API
- **Groups**: Top 43 by RiskScore = High-risk; remaining 63 = Low-risk

### Transcriptomic Analysis

- **Data**: TCGA-LIHC RNA-seq (GDC, TPM normalized)
- **DEG**: Welch two-sample t-test with Benjamini-Hochberg FDR correction
- **GSEA**: Prerank on t-statistic; MSigDB Hallmark 2020 gene sets (gseapy)

### MRI Radiomics

- **Modality**: Arterial-phase MRI (A.nrrd + tumour mask A-mask.nrrd)
- **Features**: PyRadiomics — firstorder, GLCM, GLRLM, GLSZM, GLDM, shape (≈100 features)
- **Model**: CoxnetSurvivalAnalysis (scikit-survival), l1_ratio=1.0 (pure LASSO)
- **Selection**: 5-fold CV by concordance index
